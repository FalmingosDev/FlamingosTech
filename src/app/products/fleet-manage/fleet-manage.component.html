Fleet / Employee Management System with Real Time Geo Location Tracking Facility with Analytical Reporting System

  • The most ideal solution to manage the fleet or employee movement through this application.
  • Android app in the handheld devices of the users consistently captures geo location of the device and delivers to the cloud server. An integrated web application enables the Administrators/Management to track each and every device thus the employee/ vehicle on the basis of real time data.
  • Based on the captured data this application is capable to generate various real time and periodical reports like actual position, distance travelled, location visited, duration of work, duration present in particular location etc. This application can also be used as attendance management system , which can be used as an input data for any payroll management system.
