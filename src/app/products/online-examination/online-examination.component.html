Online Examination Management System

  • The application is developed for people to take online examination by registering. The admin can set the price for each examination.
  • The questions will be set in MCQ pattern. The examination can be have specific time duration. Also user can review the answer once the examination period is over.
  • The Admin and professor group will create the question bank also super admin have the authority to review and publish the examination.
  • All test can contain any number of questions to be answered in the time specified
